# Assets

This folder stores Reliakit brand assets.

## Expected files

| File | Purpose |
|---|---|
| `reliakit-logo.png` | Horizontal wordmark logo, used in the root README header |
| `reliakit-icon.png` | Icon-only "R" mark, used for repo avatar, favicons, and docs |

## Guidelines

- Prefer optimized PNG or SVG files.
- Keep filenames stable; the root README links depend on them.
- Do not commit ZIP archives or unoptimized source exports.
- Replace files in-place when updating; do not rename or version the filenames.
